PC Requirement system

Minimum Requirements

OS Windows 7 SP1, Windows 8, Windows 8.1, Windows 10 (64bit versions only)
Processor Intel CPU Core-I5 3.2GHz or above
Memory 8 GB RAM
Graphics NVIDIA GeForce GTX 750Ti or AMD R7 260X equivalent
DirectX Version 11
Network Broadband Internet connection
Storage 15 GB available space
Sound Card DirectSound (DirectX) compatible sound card

Recommend Requirements

OS Windows 7 SP1, Windows 8, Windows 8.1, Windows 10 (64bit versions only)
Processor Intel CPU Core-I7 3770 3.4GHz or above
Memory 8 GB RAM
Graphics GeForce GTX 970 or AMD RX 560 2GB equivalent
DirectX Version 12
Network Broadband Internet connection
Storage 15 GB available space
Sound Card DirectSound (DirectX) compatible sound card
